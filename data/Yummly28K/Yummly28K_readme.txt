The dataset are crawled  from  Yummly (http://www.yummly.co/) through the Yummly API. It should be only used for non-commercial research and/or educational purposes. 
Please find the details of the dataset in the paper[1].
[1]Weiqing Min, Shuqiang Jiang, Jitao Sang, Huayang Wang, Xinda Liu, Luis Herranz .Being a Super Cook: Joint Food Attributes and Multi-Modal Content Modeling for Recipe Retrieval and Exploration, IEEE Transactions on Multimedia, 2017 (in public).
This .rar contains three parts.
(1)data_records_27638.txt: an index file  for the original recipe ids.
(2)images27638: This folder contains 27638 images.The numbers in the filename are corresponding to the number of TXT file, the user can check the original name of pictures by the numbers match.
(3)metadata27638: This folder contains the original metadata information of each recipe.
